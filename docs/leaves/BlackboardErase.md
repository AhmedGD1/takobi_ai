# BlackboardErase

Removes a key from the blackboard.

## What it does

Deletes `Key` from the blackboard, succeeding if the key existed and was removed, failing if it wasn't present.

## Properties

| Property | Description |
|---|---|
| `Key` | The blackboard key to remove. |

## Example use case

Clearing a `$target` reference once an enemy has been dealt with or is no longer relevant.
